# Todo Full Stack Web Application with Neon DB

This is a full-stack todo application that uses Neon PostgreSQL database as a service.

## Features

- Full-stack web application
- Backend built with Python/FastAPI
- Frontend built with React
- Connected to Neon PostgreSQL database
- Authentication system
- Docker support for containerization

## Setup Instructions

1. Clone the repository
2. Install dependencies for both backend and frontend
3. Set up your Neon PostgreSQL database credentials
4. Run the application

## Technologies Used

- Python (FastAPI)
- React
- PostgreSQL (Neon DB)
- Docker
- JavaScript/Node.js

## Project Structure

- `backend/` - Contains the Python FastAPI backend
- `frontend/` - Contains the React frontend
- `connect_to_neon.sh` - Script to connect to Neon DB
- `Dockerfile` - Docker configuration
- `package.json` - Node.js dependencies
